Detailed Description

Derivatives used in 3D integration steps (RK4, etc.).

Typically:

  • dx, dy, dz = velocity,
  • dvx, dvy, dvz = acceleration,
  • dwx, dwy, dwz = angular acceleration,
  • dOri = orientation derivative = 0.5 * (0, w) * q
