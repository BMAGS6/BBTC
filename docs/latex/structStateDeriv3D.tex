\doxysubsection{Detailed Description}
Derivatives used in 3D integration steps (RK4, etc.). 

Typically\+:
\begin{DoxyItemize}
\item {\ttfamily dx}, {\ttfamily dy}, {\ttfamily dz} = velocity,
\item {\ttfamily dvx}, {\ttfamily dvy}, {\ttfamily dvz} = acceleration,
\item {\ttfamily dwx}, {\ttfamily dwy}, {\ttfamily dwz} = angular acceleration,
\item {\ttfamily d\+Ori} = orientation derivative = 0.\+5 \texorpdfstring{$\ast$}{*} (0, w) \texorpdfstring{$\ast$}{*} q 
\end{DoxyItemize}
