\doxysubsection{Detailed Description}
Prototype interface for the 6-\/DOF ballistic solver, now with actual drag table usage. 

This file\+:
\begin{DoxyItemize}
\item Loads a CSV drag table into a global array,
\item Precomputes the finer-\/resolution lookup table,
\item Initializes the environment (weather, altitude, muzzle conditions),
\item Initializes projectile specs (mass, inertia, aero, etc.),
\item Runs the 6-\/DOF solver in a loop until the projectile hits the ground or times out. 
\end{DoxyItemize}
