\doxysubsection{Detailed Description}
This module handles\+:
\begin{DoxyItemize}
\item Defining standard drag models (G1, G7, etc.),
\item Parsing CSV tables of Mach/\+Cd values,
\item Interpolating drag coefficients at arbitrary Mach numbers,
\item Precomputing fast lookup tables for runtime efficiency. 
\end{DoxyItemize}

Linearly interpolates a drag coefficient from the global lookup table. 

Given a Mach number, this function searches for two nearby entries in the {\ttfamily lookup\+Table\mbox{[}\mbox{]}} and performs linear interpolation. If {\ttfamily mach} is outside the table range, it clamps to the first or last available value.


\begin{DoxyParams}[1]{Parameters}

Linearly interpolates a drag coefficient from the global lookup table.

Internally does a binary search to find the bracketed region. Then performs linear interpolation\+:

\[
  \mathrm{Cd}(mach) = \mathrm{Cd}(m_1) + \frac{mach - m_1}{m_2 - m_1}\, \bigl(\mathrm{Cd}(m_2) - \mathrm{Cd}(m_1)\bigr).
\]

\begin{DoxyParagraph}{Complexity}
$ \mathcal{O}(\log n) $ due to binary search.
\end{DoxyParagraph}
