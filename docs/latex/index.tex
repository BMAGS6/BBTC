BBTC is written entirely in C, and is designed with deep educational value in mind, offering\+:
\begin{DoxyItemize}
\item Advanced 6DOF modeling with quaternion rotation
\item Support for all major drag models (G1, G2, G5, G6, G7, G8, GL, GS, GI, RA4)
\item Realistic atmosphere and environmental effects
\item Detailed .csv (spreadsheet format---comma-\/separated values) logging and simulation output
\item Fully static memory model with no dynamic allocation (no {\ttfamily malloc} , {\ttfamily realloc} , {\ttfamily calloc} , etc.)
